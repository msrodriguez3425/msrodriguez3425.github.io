Summary: Lattitude vs. X

Visualizations

The aim of this project was to use APIs prove something that is usually considered to be obvious: How does the weather change as one moves from the poles towards the equator?

I considered 4 variables in order to answer this question: Wind Speed, Temperature, Humidity, and Cloud Cover.

I generated a list of 500 cities across the globe and extracted each of the above variables corresponding to each city, along with their lattitudes, from the open weather API. I then created a plot for each variable against lattitude, where each point represents a city.
